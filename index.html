  • Difference between 'switch' and 'if/else'?
    • 'Switch' - has a set of cases available and will select one of those cases based on what has been inputed.
    • 'If/else' - has a set of possible scenarios and each scenario will be checked until one matches the criteria of one of the if statements and then it will run the code of that statement.
  • When to us '=' vs '==' vs '.equals'?
    • '=' is declarative. It is used to set a variable equal to something else.
    • '==' is a comparison. You can use this to check if something equals something else.
    • '.equals' is used to compare things inside String variables.
  • What is 'Math' in 'Math.random()'?
    • 'Math' is a built in java function.
    • '.random()' returns a Double.
    • The 'M' in 'Math' is capitalized to show that it is a built in class.
  • What does it mean to 'cast a double to an int'?
    • You remove everything to the right of the decimal. 9.99 -> 9.
    • You do this in various 'Math.random()' scenarios where you want to find a random number between ## and ##.
  • What is the difference between 'while' and 'do while' loops?
    • 'while' sets the contingency for statement first and only runs the code after if that contingency is met.
    • 'do while' sets the contingency last so the code after 'do' will always be run at least one time.
    • 'do while' has the benefit of running the code inside at least once before leaving that part of the code, or continuing based on if the 'while' part is met or not. Sometimes what you're checking in the condition is something you won't know until after you've run the code once.
  • What happens when you include 'import java.util.Scanner;' in your code?
    • It imports a scanner into your code so you can take inputs from a user.
    • The functionality comes from a java utility database.
    • 'java' in 'java.util.Scanner;' is the head directory where we can locate the util directory and then actually import 'Scanner.'
  • What is 'void' in a method declaration?
    • 'void' means that the code will not return any information once it is finished.
    • This differs from a return type because a return type would return information once it is finished
  • What is the difference between a 'parameter' and an 'argument?'
    • A 'parameter' is a variable in a method definition.
    • When that method is called, the 'arguement' is the data that moves into the 'parameter.'
